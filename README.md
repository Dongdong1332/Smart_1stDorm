# smartDorm
##### ##### #####
🟫 使用的材料
1. 超音波距離感測器 HC-SR04
2. 繼電器模組
3. Arduino UNO
4. PC
##### ##### #####

💡💡💡💡💡💡💡💡💡💡💡💡💡💡💡

安裝一個自動化控制宿舍桌上的檯燈
這是一個我新發現的物聯網實現方法之一
也可以說是一種帶有 SoC 軟硬體整合概念的小專題
可以透過Arduino與電腦內的Python透過序列阜進行溝通

========================================
🟩 Arduino負責硬體：
1. 持續讀取超音波感測器資料
2. 持續等待使否有指令需控制繼電器

========================================
🟥 電腦Python負責軟體：
1. 判斷式都寫在這邊
2. 整合拿到的資料，做出控制的決策
